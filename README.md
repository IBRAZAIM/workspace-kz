# WorkSpace.kz — Платформа аренды кабинетов

Проект представляет собой полноценную систему для поиска, бронирования и управления почасовой арендой рабочих пространств.

## 🚀 Быстрый запуск

Для запуска проекта на локальном компьютере выполните следующие шаги:

### 1. Требования
Убедитесь, что у вас установлен **Node.js** (версия 16 или выше). Скачать можно на [nodejs.org](https://nodejs.org/).

### 2. Установка зависимостей
Откройте терминал (командную строку) в папке проекта и выполните:
```bash
npm run setup
```
Эта команда установит все необходимые компоненты для сервера и интерфейса.

### 3. Запуск приложения
После установки запустите проект командой:
```bash
npm start
```
Приложение будет доступно по адресу: **http://localhost:5500**

## 🛠 Технологии
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Backend:** Node.js, Express.
- **База данных:** SQLite (файл `backend/database.sqlite` создается автоматически).

## 📁 Структура проекта
- `/backend` — серверная часть (API, база данных, авторизация).
- `/index.html` — главная страница каталога.
- `/admin.html` — панель администратора.
- `/dashboard.html` — личный кабинет пользователя.
- `/main.js`, `/api.js`, `/auth.js` — основная логика фронтенда.
